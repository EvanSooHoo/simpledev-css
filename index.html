simpledev.css

The goal of simpledev.css is to create a mostly classless CSS framework. We are starting off with working on a handful of components and will add the classless code later.

Navbar

Brand
  • Home
  • Pricing
  • About
  • Contact

    <nav class="navbar">
      <div>
        <a href="">Brand</a>
        <button>Menu</button>
      </div>
      <ul>
        <li><a href="">Home</a></li>
        <li><a href="">Pricing</a></li>
        <li><a href="">About</a></li>
        <li><a href="">Contact</a></li>
      </ul>
    </nav>

The navbar uses the navbar class. You can add more links by adding more li elements.

Non-sticky navbar


    .navbar {
      position: static;
    }

Add this code to your custom CSS file to remove the sticky feature from the navbar.

Navbar Message

Test navbar message.

    <div class="navbar-message">Test navbar message.</div>
    <nav class="navbar">...</nav>

You can add a div with the navbar-message class above your nav element with the navbar class. All it does is add a background color and center the text, but you can customize it in your custom CSS file.

Container


    <div class="container">

    </div>

Use the container class with a div element to apply a max width and center the contents.

Buttons

Anchor 

    <a href="" class="button">Anchor</a>
    <button type="button">Button</button>
    <input type="button" value="Input">
    <input type="submit" value="Submit">
    <input type="reset" value="Reset">

You can use the button class on an a element to make it look like a button. The other elements that look like buttons do not need a class.

Footer

© 2023 Business Name. 

    <footer class="footer">
      <small class="">© 2023 Business Name.</small>
    </footer>

You can add a footer to your site using a footer element with the footer class. We use a footer class since the footer element can be used in different contexts. All it does is add a background color, some padding, and centers the text.
