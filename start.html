Quick Start

Add CSS

Add the following link element to add simpledev.css to your HTML file. This element should go inside your head element. Our CSS file includes modern-normalize.css to help with any browser compatibility issues. You can also download the latest version of simpledev.css if you want to host the file yourself.


      <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/dist/simpledev.css">

Add JS

If you're going to use the navbar, make sure you also include the CDN link for our JS file. It can go inside the head element since it has the defer attribute. You can also download the latest version of our JS file if you want to host the file yourself.

<script src="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/dist/navbar.js" defer></script>

Dark Mode

We are experimenting with adding dark mode support to simpledev.css. To add dark mode to your HTML file, just copy the link element below and put it underneath the link element from above. For now the plan is to keep it as a separate link element since not everyone wants dark mode. You can also download the latest version of dark.css if you want to host the file yourself.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/dist/dark.css">

Permanent Dark Mode

If you want your site to always have a dark theme, download the file using the link above and save it inside your project folder. Then delete the media query wrapping all of the CSS code (don't forget to delete the curly brace at the end).


      /* Delete the media query */
      @media (prefers-color-scheme: dark) {
        /* Keep everything inside the media query */
      }
      /* Don't forget to delete the closing brace */

Lastly, you can link to the CSS file in your HTML file. In the example below, the file is inside a folder called css.

<link rel="stylesheet" href="css/dark.css">

Add custom CSS

Lastly, we encourage adding your own custom CSS using your own CSS file. You can technically just stick with the default styles, but we recommend adding your own colors and fonts to make your site unique.

For example, if you have a file called custom.css inside your css folder, you should add the following link element after the other two link elements we listed above. You can always name the file something else or put it in a different folder (just make sure the path is right).

<link rel="stylesheet" href="css/custom.css">

Basic Template

You can copy and paste this into an empty HTML file to quickly get started. This template also includes the dark mode link element. If you don't want dark mode, you can simply delete that link element. If you want to add custom CSS, you'll have to add the link element for that yourself.


      <!DOCTYPE html>
      <html lang="en">
      <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">

        <title>Template</title>

        <!-- CSS -->
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/dist/simpledev.css">
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/dist/dark.css">

        <!-- JS -->
        <script src="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/dist/navbar.js" defer></script>
      </head>

      <body>
        <h1>Template</h1>
      </body>
      </html>

Advanced Template

Copy and paste this code into an empty HTML file if you want a template that already has a navbar, container, and footer. This template also includes the dark mode link element. If you don't want dark mode, you can simply delete that link element.


      <!DOCTYPE html>
      <html lang="en">
      <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">

        <title>Advanced Template</title>

        <!-- CSS -->
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/dist/simpledev.css">
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/dist/dark.css">

        <!-- JS -->
        <script src="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/dist/navbar.js" defer></script>
      </head>

      <body>
        <nav class="navbar">
          <div>
            <a href="">Brand</a>
            <button type="button">Menu</button>
          </div>
          <ul>
            <li><a href="" class="active">Home</a></li>
            <li><a href="">Pricing</a></li>
            <li><a href="">About</a></li>
            <li><a href="">Contact</a></li>
          </ul>
        </nav>

        <div class="container">
          <h1>Advanced Template</h1>
        </div>

        <footer class="footer">
          <ul class="footer-links">
            <li><a href="">Facebook </a></li>
            <li><a href="">Twitter</a></li>
            <li><a href="">Instagram</a></li>
            <li><a href="">LinkedIn</a></li>
            <li><a href="">YouTube</a></li>
          </ul>
          <small class="">© 2023 Business Name.</small>
        </footer>
      </body>
      </html>
