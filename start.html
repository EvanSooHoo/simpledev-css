Quick Start

Add CSS

These components were developed using modern-normalize.css. It's recommended that you include modern-normalize.css in your project if you're going to use this framework. Then you can add the CDN link for simpledev.css from jsDelivr. These elements should go inside your head element.


      <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/modern-normalize@2.0.0/modern-normalize.min.css">
      <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/css/simpledev.css">

Add JS

If you're going to use the navbar, make sure you also include the CDN link for our JS file. It should go just above the closing body tag.

<script src="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/js/navbar.js"></script>

Add custom CSS

Lastly, we encourage adding your own custom CSS using your own CSS file. You can technically just stick with the default styles, but we recommend adding your own colors and fonts to make your site unique.

For example, if you have a file called custom.css inside your css folder, you should add the following link element after the other two link elements we listed above. You can always name the file something else or put it in a different folder (just make sure the path is right).

<link rel="stylesheet" href="css/custom.css">

Basic Template

You can copy and paste this into an empty HTML file to get started. If you want to add custom CSS, you'll have to add the link element for that yourself.


      <!DOCTYPE html>
      <html lang="en">
      <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">

        <title>Template</title>

        <!-- CSS -->
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/modern-normalize@2.0.0/modern-normalize.min.css">
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/css/simpledev.css">
      </head>

      <body>
        <h1>Template</h1>

        <!-- JS -->
        <script src="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/js/navbar.js"></script>
      </body>
      </html>

Advanced Template

Copy and paste this code into an empty HTML file if you want a template that already has a navbar, container, and footer.


      <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  <title>Advanced Template</title>

  <!-- CSS -->
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/modern-normalize@2.0.0/modern-normalize.min.css">
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/css/simpledev.css">
</head>

<body>
  <nav class="navbar">
    <div>
      <a href="">Brand</a>
      <button type="button">Menu</button>
    </div>
    <ul>
      <li><a href="" class="active">Home</a></li>
      <li><a href="">Pricing</a></li>
      <li><a href="">About</a></li>
      <li><a href="">Contact</a></li>
    </ul>
  </nav>

  <div class="container">
    <h1>Advanced Template</h1>
  </div>

  <footer class="footer">
    <small class="">© 2023 Business Name.</small>
  </footer>

  <!-- JS -->
  <script src="https://cdn.jsdelivr.net/gh/simpledevio/simpledev-css@latest/js/navbar.js"></script>
</body>
</html>
